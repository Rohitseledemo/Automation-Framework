# Selenium WebDriver Automation Framework  
A modular, robust and scalable automation testing framework built with Selenium WebDriver and Java. 
Designed for efficient test creation, execution, and reporting across multiple browsers and platforms.

# Key Features
- Implements the Page Object Model (POM).
- Singleton Pattern for Browser Initialization.
- Parallel test execution with TestNG.
- Generates detailed HTML reports.
- Failed Test Screenshots.
- Data-Driven Testing.
- Reusable utilities like Dynamic Wait Mechanisms, ReportGenerator, Error Handling, Retry Mechanism, Dropdown Handling,
  Common table re-usuables etc.
- Integrates with CI/CD tools like BitBucket Pipelines.

# Prerequisites
- Java (e.g., JDK 8 or higher)
- Selenium Webdriver (4.10 or higher)
- TestNG (7.4.0 or higher)
- browserstack-local-java (1.0.6 or higher)
- browserstack-java-sdk (1.10.7)
- Extent Reports (5.1 or higher )
- BrowserStack Account (for remote testing)
- Maven (for dependency management)
- Browser drivers (e.g., ChromeDriver, firefox etc.)

# Setup Instructions
- Clone the Repository
- Install Dependencies
- Configure Browser Drivers 
  * Download browser drivers (e.g., ChromeDriver, GeckoDriver).
  * Place them in a directory (e.g., src/main/resources/chromeDriver).
  * Update the config.properties file with the path to your drivers.
